# Kasper - Creative One Page Template 🎨

This is the second project in my Front-End development journey. **Kasper** is a professional, modern, and creative one-page portfolio template designed for agencies and creative professionals. It showcases advanced CSS techniques and a deep understanding of complex web layouts.

---

## 🔗 Live Demo
You can view the live preview of the website here:
> **[View Live Site ✨](https://ibrahim-hamdouna.github.io/kasper-template/kasper.html)**

---

## 🧠 Technical Highlights & Skills
* **Advanced Layouts**: Utilizing **CSS Grid** for the portfolio and services sections to manage complex structures.
* **Precise Positioning**: Using **Flexbox** for navigation, buttons, and centering elements.
* **Responsive Web Design**: Optimized for all screen sizes (Mobile, Tablet, and Desktop) using CSS Media Queries.
* **Modern UI Elements**: Custom buttons, hover effects, and Google Fonts integration for a premium user experience.

## 🛠 Built With
* **HTML5**: For clean and semantic web structure.
* **CSS3**: Using Grid, Flexbox, custom variables, and animations.
* **FontAwesome**: For scalable and high-quality vector icons.
* **Google Fonts**: Specifically using "Open Sans" for a professional look.

## 📁 Project Structure
The project is organized with best practices in mind:
* `kasper.html` - The main entry point of the project.
* `css/` - Optimized directory for stylesheets (Lowercase).
* `images/` - Optimized directory for visual assets (Lowercase).
* `webfonts/` - Essential files for iconography.
